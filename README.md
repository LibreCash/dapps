## LibreBank dApps

LibreBank dApps (decentalized apps) repository.

### dApps List:
1. ETH Blockchain Reports dApp.
2. Fund StatusPage.
3. Contract StatusPage.
4. Deposit (Draft).
5. DAO (Draft).
6. Loans (Draft).
7. Faucet (for testing in Rinkeby network, draft)

## Getting Started
```sh
git clone https://github.com/LibreCash/dapps && cd dapps
npm install
npm run dev
```
